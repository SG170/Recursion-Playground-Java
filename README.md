# Recursion Playground (Java)

A simple **menu-driven Java console application** designed to demonstrate core **recursion-based DSA problems**.  
This project helps in understanding recursion concepts such as **base cases, recursive calls, and problem reduction**.

---

##  Features

- Factorial using recursion  
- Fibonacci number calculation  
- Sum of digits of a number  
- Power of a number  
- Reverse a number  
- GCD (Greatest Common Divisor)  
- Interactive console menu  

---

## Technologies Used

- Java  
- DSA (Recursion)  
- VS Code  
- Git & GitHub
